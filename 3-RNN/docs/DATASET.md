# Dataset

Este proyecto requiere un corpus con 60+ funciones en C, con estilo consistente.

## Donde colocarlo

- Archivo recomendado: `dataset/funciones.c`
- Puedes usar varios archivos si prefieres, pero empieza con uno.

## Guia rapida

1. Reune tus funciones en C.
2. Verifica que todas tengan formato consistente.
3. Copialas dentro de `dataset/funciones.c`.
4. No mezcles otros lenguajes en el mismo archivo.

## Reglas recomendadas

- Nombres de funciones con un mismo estilo (snake_case o camelCase).
- Misma estructura de llaves y sangria.
- Comentarios opcionales, pero consistentes.
- Evita macros grandes o headers extensos.
